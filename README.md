# 🚀 LiteLoader

**LiteLoader** is a high-performance, minimalist video downloader built with Python. It allows users to download content from YouTube, Instagram, and many other platforms with a single click.

---

## ✨ Key Features
- **Modern UI:** Built with `CustomTkinter` for a sleek dark-mode look.
- **High Quality:** Automatically fetches the best quality available.
- **Cross-Platform:** Full support for Windows, macOS, and Linux.
- **Fast & Lightweight:** Minimal resource usage.

---

## 🛠️ Prerequisites

To merge high-quality (1080p+) video and audio, **FFmpeg** is required:
- **macOS:** `brew install ffmpeg`
- **Windows:** Download from [ffmpeg.org](https://ffmpeg.org/download.html)
